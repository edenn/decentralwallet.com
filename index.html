Colored Coins

A Bitcoin 2.0 Protocol that can allow you to issue tokens on top of the Bitcoin Blockchain without the need to buy a general metacoin to have acesss to the tokens.

Multisignature

The ability to issue and import P2SH addresses for escrow and other contract agreements.

Peer to Peer

Using Webrtc we can allow for trust free peer to peer communication without a need for a backend.
